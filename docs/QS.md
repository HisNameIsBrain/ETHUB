### Intake
- Verify identity, contact, device, serial
- Photos of condition
- Issue description
- Assign order number

### Diagnosis
- Run tests; capture findings
- Decide parts required; estimate time/cost
- Set status: diagnosis or awaiting_parts

### Parts
- Create vendor order with ETA
- Notify delay if ETA slips

### Repair
- Log parts used and steps taken
- Photos if helpful

### Quality Check
- Functional test original issue
- Regression test main functions
- Clean device
- Fastenings intact
- Final photos
- Set status: ready

### Completion
- Handover/pickup or ship
- Collect payment
- Close with delivered status
